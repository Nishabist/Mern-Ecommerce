# MERN Stack E-commerce
 Ecommerce website built with the MERN stack with Redux for state management and Tailwind , Ant design for designing UI.


### Features
- **User Authentication**
- **Forget Password**
- **Product Add**
- **Product Remove**
- **Product Categorization** 
- **Add To Cart and Checkout**
- **Payment**: Strip
